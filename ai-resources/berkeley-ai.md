# UC Berkeley AI Tools

Campus-licensed AI tools available to all Berkeley affiliates.

## Campus AI Sandbox (BETA)

- **URL**: [ai.berkeley.edu](https://ai.berkeley.edu/tools-services/licensed-generative-ai-tools)
- **Auth**: CalNet login
- **Interface**: LibreChat-based browser interface
- **Models**: Claude, GPT, Gemini (browser-only, no API access)
- **Data protection**: UC-negotiated terms, data not used for training

## Google Gemini App

- Available via CalNet authentication
- Gemini Pro model
- Data-protected under UC agreement

## Zoom AI Companion

- Available to all Berkeley Zoom users
- Meeting summaries, chat assistance

## Limitations

These are browser-only tools with no programmatic API access. For API access to the same models, use [CBORG](cborg.md) instead.
